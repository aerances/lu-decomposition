1. Mathematical Discussion

LU Decomposition is a matrix factorization technique in numerical methods where a square matrix \(A\) is decomposed into the product of two triangular matrices:

$$ A = LU $$

where:

  • \(L\) is a Lower Triangular Matrix containing values below the diagonal.
  • \(U\) is an Upper Triangular Matrix containing values above the diagonal.

In the Doolittle Method, all diagonal entries of the lower triangular matrix are equal to 1:

$$ l_{ii} = 1 $$

The main goal of LU decomposition is to simplify complex matrix operations such as solving systems of linear equations, matrix inversion, and determinant computation.

Instead of directly solving:

$$ AX = B $$

the matrix \(A\) is first decomposed into:

$$ A = LU $$

which transforms the original equation into:

$$ LUX = B $$

Let:

$$ UX = Y $$

then the problem becomes:

$$ LY = B $$

This allows the solution to be obtained using two simpler stages:

  1. Forward Substitution to solve \(LY = B\)
  2. Backward Substitution to solve \(UX = Y\)

The Doolittle algorithm computes the entries of the upper matrix row-by-row and the lower matrix column-by-column using the following formulas:

Upper Triangular Matrix Formula:

$$ u_{ij} = a_{ij} - \sum_{k=1}^{i-1} l_{ik}u_{kj} $$

Lower Triangular Matrix Formula:

$$ l_{ij} = \frac{ a_{ij} - \sum_{k=1}^{j-1} l_{ik}u_{kj} }{ u_{jj} } $$

Applications of LU Decomposition

  1. Solving Linear Systems: Efficiently solves simultaneous equations in engineering and scientific computations.
  2. Matrix Inversion: LU decomposition simplifies finding the inverse of a matrix.
  3. Determinant Evaluation: The determinant of matrix \(A\) can be obtained from the product of the diagonal elements of matrix \(U\).
  4. Computer Simulations: Frequently used in numerical analysis, finite element methods, and machine learning computations.
