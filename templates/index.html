1. Mathematical Discussion

LU Decomposition factorizes a given square coefficient matrix \(A\) into a product of two companion triangular matrices: a Lower Triangular Matrix (\(L\)) and an Upper Triangular Matrix (\(U\)), such that:

$$A = LU$$

In the Doolittle algorithm variant, the diagonal elements of matrix \(L\) are equal to 1:

$$l_{ii} = 1 \quad \text{for all } i$$

Doolittle's method provides an alternative way to factorize \(A\) into an LU decomposition directly without going through Gaussian Elimination.

  • Upper Matrix Elements (\(U\)): $$u_{ij} = a_{ij} - \sum_{k=0}^{i-1} l_{ik}u_{kj}$$
  • Lower Matrix Elements (\(L\)): $$l_{ij} = \frac{ a_{ij} - \sum_{k=0}^{j-1} l_{ik}u_{kj} }{ u_{jj} }$$
